# Android SDK setup (Windows)

Typical SDK path:

C:\Users\<USER>\AppData\Local\Android\Sdk

Environment variables (either works):

- ANDROID_SDK_ROOT
- ANDROID_HOME

Create local.properties at the repo root:

sdk.dir=C\:\\Users\\<USER>\\AppData\\Local\\Android\\Sdk

Notes:
- local.properties should not be committed.
- You can install the SDK via Android Studio or the Android command line tools.
